# hipsteralert
Lets you know when the [Prince Charles Cinema](www.princecharlescinema.com) has anything good on

#### installation
```sh
npm install
npm run start
```

#### usage
```sh
$ curl localhost:3000/films
{"HIGH-RISE":[{"time":"1:00pm","slug":"15852"}]..
```
